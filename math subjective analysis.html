Quick Win Patterns

These recurring problem templates appear across 10+ papers each. Learn the template once — solve any variant.

01
Tangent Pair at 60° — The Universal Construction
Draw a pair of tangents to a circle of radius r cm which are inclined at 60°. This is identical across 15+ papers — only the radius changes (3, 4, 5, 6 cm).
📍 Appears in: JK-82, NK-8, MP-11, K-8, RJ-82, G-12, Y-8, M-8, and 7+ more
💡 Trick: The angle between tangents is 60°, so the angle at centre = 120°. Draw radius at 120° to each other, then erect perpendiculars.
02
Quadratic Polynomial Graph → Zeroes
Draw graph of p(x) = x² ± bx ± c, find where it crosses x-axis. The zeroes are always integers (−4 to 4 range). Same structure in every paper.
📍 Appears in: NK-8, V-11, R-8, SK-82, MP-11, RM-82, Y-8, M-8, M-08, n-11 and 10+ more
💡 Trick: Always make a table for x = −4, −3, −2, −1, 0, 1, 2, 3, 4. Plot and join smoothly. Zeroes = x-intercepts.
03
Trisection of a Line Segment
Find the co-ordinates of points of trisection of a segment joining (x₁,y₁) and (x₂,y₂). Uses Section Formula at ratios 1:2 and 2:1. Appears in 10+ papers.
📍 Appears in: K-8, RJ-82, RM-82, SK-82, PR-82, V-11, PF-2026, M-08 and more
💡 Trick: P divides in 1:2 → P = ((x₁+2x₂)/3, (y₁+2y₂)/3). Q divides in 2:1 → Q = ((2x₁+x₂)/3, (2y₁+y₂)/3).
04
Trig Identity Proof: secθ/tanθ variations
Prove that (expression with sec, tan, cosec, cot) = (simpler form). Most proofs boil down to converting everything to sin and cos, or using the identity 1+tan²θ = sec²θ.
📍 Appears in: JK-82, K-8, R-8, MP-11, RM-82, MF-2026, PF-2026 and every other paper
💡 Master Template: Take LHS → convert to sin/cos → simplify → match RHS. Or use a²−b² = (a+b)(a−b) for identity expressions.
05
Two Observers / Two Poles Heights & Distances
Two people on either side of a tower/building observe its top at elevation angles θ₁ and θ₂. Find the distance between them (or heights).
📍 Appears in: JK-82, K-8, R-8 (implicit), SK-82 (variant), NK-8b, MP-11, MF-2026 and 8+ more
💡 Setup: Let tower height = h. Observer 1 at distance d₁ → tan θ₁ = h/d₁. Observer 2 at distance d₂ → tan θ₂ = h/d₂. Answer = d₁ + d₂.
06
If cosecθ+cotθ = k, prove sinθ/cosθ = ...
Given secθ+tanθ = P (or cosecθ+cotθ = k), express all other trig ratios in terms of P or k. The answer always uses (P²−1)/(P²+1) or (P²+1)/2P.
📍 Appears in: JK-82, J-8, MP-11, ST-82, PF-2026, DP-13, MF-2026
💡 Key identity: If secθ+tanθ = k → secθ−tanθ = 1/k → secθ = (k+1/k)/2, sinθ = (k²−1)/(k²+1).
07
Statistics Median — CF Table Method
Given a frequency distribution, find Median using cumulative frequency table. Always: (1) Find n/2, (2) Identify median class, (3) Apply formula. Same steps every time.
📍 Appears in: NK-8, K-8, R-8, V-11, RJ-82, SK-82, PR-82, RM-82, Y-8 and more
💡 Formula: M = l + [(n/2 − cf)/f] × h. The class intervals are usually 10-wide. cf = cumulative frequency BEFORE median class.
08
Prove Irrational (√a + √b or 2√a − √b)
Prove that 3√5+√7, √3−√5, 2√7−√5 etc. are irrational. Every proof uses the same contradiction template — assume it's rational, derive a contradiction.
📍 Appears in: JK-82, V-11, K-8, S-11, RJ-82, SK-82, M-8, M-08, RM-82 and more
💡 Template: Assume p/q = 2√a+√b → isolate the irrational → show LHS is irrational while RHS is rational → contradiction. Done.

Must-Know Formulas

Only the formulas that actually appear in exams. Memorise these — nothing else.

// Statistics
Median (Grouped)
M = l + [(n/2 − cf) / f] × h l = lower boundary of median class n = total frequency cf = cumulative freq before class f = frequency of median class h = class width
Mode (Grouped)
Mode = l + [(f₁−f₀) / (2f₁−f₀−f₂)] × h f₁ = freq of modal class f₀ = freq of class before modal f₂ = freq of class after modal l, h = lower boundary, class width
Mean — Step Deviation
Mean = A + (Σfᵢuᵢ / Σfᵢ) × h A = assumed mean uᵢ = (xᵢ − A) / h h = class width
// Coordinate Geometry
Section Formula
P = ((mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n)) For trisection: P₁: ratio 1:2 → ((x₁+2x₂)/3, (y₁+2y₂)/3) P₂: ratio 2:1 → ((2x₁+x₂)/3, (2y₁+y₂)/3)
Area of Triangle & Collinearity
Area = ½|x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)| Collinear ⟺ Area = 0 Centroid = ((x₁+x₂+x₃)/3, (y₁+y₂+y₃)/3)
Distance & Midpoint
Distance = √[(x₂−x₁)² + (y₂−y₁)²] Midpoint = ((x₁+x₂)/2, (y₁+y₂)/2)
// Trigonometry — Core Identities
Pythagorean Identities
sin²θ + cos²θ = 1 1 + tan²θ = sec²θ 1 + cot²θ = cosec²θ Derived: sec²θ − tan²θ = 1 cosec²θ − cot²θ = 1
secθ+tanθ = k → sinθ
If secθ + tanθ = k, then: secθ − tanθ = 1/k secθ = (k² + 1) / 2k tanθ = (k² − 1) / 2k sinθ = (k² − 1) / (k² + 1) cosθ = 2k / (k² + 1)
Ratio Conversions
sinθ in terms of tanθ: sinθ = tanθ / √(1+tan²θ) cosθ in terms of cotθ: cosθ = cotθ / √(1+cot²θ) tanθ in terms of sinθ: tanθ = sinθ / √(1−sin²θ)
Triangle Angle Identity
In △ABC, A+B+C = 180° ∴ tan((A+B)/2) = cot(C/2) cot((A+B)/2) = tan(C/2) sin((A+B)/2) = cos(C/2) Used for proving trig results in triangles
// Arithmetic Progression
AP Essentials
nth term: aₙ = a + (n−1)d Sum of n terms: Sₙ = n/2[2a + (n−1)d] = n/2[a + l] (l = last term) nth from end: aₙ = l − (n−1)d
// Mensuration (Most Common)
Volume Conversions (melting/recasting)
Sphere V = (4/3)πr³ Cylinder V = πr²h Cone V = (1/3)πr²h Hemisphere V = (2/3)πr³ Key: V₁ = n × V₂ (n = number of pieces)
Surface Areas
Sphere: 4πr² Cylinder: CSA = 2πrh, TSA = 2πr(r+h) Cone: CSA = πrl, l = √(r²+h²) Hemisphere: CSA = 2πr², TSA = 3πr²
// Circles & Constructions
Tangent Theorems
Tangent length from external pt P: PT = √(d² − r²) where d = distance from centre, r = radius Tangents from same external pt are equal. Radius ⊥ Tangent at point of contact.
Tangent at 60° — Construction Key
Angle between tangents = 60° → Angle at centre = 120° → Draw radii making 120° angle → Construct 90° at each radius end → Two perpendiculars meet = tangent pair

Common Pitfalls

The top mistakes students make — and exactly how to avoid them.

1. Median Formula: Wrong "cf"
✗ Using the frequency of the class before, not the cumulative frequency
✓ cf = sum of ALL frequencies BEFORE the median class
Build a full cumulative frequency column first. Then find n/2 and locate the median class. cf is the row ABOVE the median class row.
2. Trisection: Confusing ratio order
✗ Using 1:2 for both trisection points
✓ P₁ divides AB in 1:2 (closer to A), P₂ divides AB in 2:1 (closer to B)
The point closer to A uses ratio 1:2. The point closer to B uses ratio 2:1. Always draw a diagram to confirm which is which.
3. Trig Identity: Wrong LHS manipulation
✗ Cross-multiplying or moving terms across the "=" in a proof
✓ Work only on ONE side (usually LHS) until it equals RHS
Never modify both sides simultaneously in a proof. Start from LHS, simplify to get RHS. If stuck, start from RHS and work backwards (but show it clearly).
4. Heights & Distances: Elevation vs. Depression
✗ Using tan for angle of depression from the wrong triangle vertex
✓ Angle of depression from top = angle of elevation from bottom (alternate angles)
Always draw the horizontal line at the observer's level. The angle of depression is measured from this horizontal, not from the vertical.
5. Mode Formula: f₀ and f₂ mix-up
✗ Mixing up which is the preceding and succeeding class frequency
✓ f₀ = frequency of class just BEFORE modal class; f₂ = class just AFTER
First identify the modal class (highest frequency). Then f₀ is the row above it, f₂ is the row below it in your frequency table.
6. Graph of Polynomial: Curve vs. Lines
✗ Drawing straight lines through the plotted points
✓ Always draw a smooth parabolic (U-shaped) curve through all points
Plot at least 7 points (x = −3 to 3). Join with a smooth curve, not a dot-to-dot polygon. The zeroes must be clearly readable from the x-axis intersection.

Quick Reference Cheat Sheet

Identify the question type → follow the steps. Printable.

📊
Statistics — Median
01
Add a Cumulative Frequency (cf) column to the table
02
Find n/2 (half of total frequency)
03
Find the first cf that is ≥ n/2 → that row is the median class
04
Apply: M = l + [(n/2 − cf)/f] × h
05
cf = cumulative frequency of the row ABOVE median class
📊
Statistics — Mode
01
Find the class with the highest frequency → modal class
02
Note: f₁ (modal class), f₀ (class before), f₂ (class after), l (lower boundary), h (class width)
03
Apply: Mode = l + [(f₁−f₀)/(2f₁−f₀−f₂)] × h
🔢
Trig Proof — Identity
01
Write LHS = ... and work only on left side
02
Convert everything to sin and cos (or use the identity directly)
03
Key swaps: sec²θ = 1+tan²θ, cosec²θ = 1+cot²θ
04
Try a²−b² = (a+b)(a−b) for difference-of-squares expressions
05
Simplify until you get = RHS. Write "∴ LHS = RHS" at the end.
📐
Heights & Distances
01
Draw a diagram first — always. Mark the known angles and distances.
02
Identify right triangles. Apply: tan θ = opposite/adjacent
03
Two observers setup: h = d₁×tan θ₁ = d₂×tan θ₂
04
Use tan 30° = 1/√3, tan 45° = 1, tan 60° = √3
05
Rationalise the denominator in final answer
📈
Polynomial Graph (6 Marks)
01
Make a table for x = −4, −3, −2, −1, 0, 1, 2, 3, 4
02
Calculate p(x) for each x value
03
Plot points on graph paper. Label axes and scale clearly.
04
Join with a smooth U-shaped curve (never straight lines)
05
Zeroes = points where curve crosses the x-axis. Read and state them.
📍
Trisection — Section Formula
01
Given A(x₁,y₁) and B(x₂,y₂)
02
P (closer to A, ratio 1:2): ((x₁+2x₂)/3, (y₁+2y₂)/3)
03
Q (closer to B, ratio 2:1): ((2x₁+x₂)/3, (2y₁+y₂)/3)
04
Verify: midpoint of P and Q should be the midpoint of AB
🔵
Tangent Pair at 60° (Construction)
01
Draw circle of given radius. Mark centre O.
02
Draw a diameter. From O, draw another radius at 120° to the first.
03
At each radius end (on circumference), erect a perpendicular to the radius.
04
The two perpendiculars meet at point P — these are the tangents.
05
Verify angle at P = 60° with a protractor (for checking)
∞
Irrational Proof Template
01
State: "Assume √a + √b is rational"
02
Write: √a + √b = p/q where p,q are integers, q≠0
03
Isolate one surd: √a = p/q − √b
04
LHS (√a) is irrational, RHS = rational − irrational = irrational. But we assumed both sides are rational — contradiction!
05
Conclude: Our assumption is wrong. ∴ √a + √b is irrational.
